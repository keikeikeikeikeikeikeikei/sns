# サーバー起動手順 & テストアカウント情報

このプロジェクトをローカルで起動するための手順と、動作確認用のアカウント情報です。

## 1. サーバーの起動

ターミナルを2つ開き、それぞれで以下のコマンドを実行してください。

### ① バックエンド (PHP) の起動
APIサーバーを立ち上げます。

```powershell
cd c:\Users\k1111\OneDrive\デスクトップ\sns2\backend
php -S localhost:8080 -t public
```

### ② フロントエンド (Vue.js) の起動
画面を表示するためのサーバーを立ち上げます。

```powershell
cd c:\Users\k1111\OneDrive\デスクトップ\sns2\frontend
npx vite
```

---

## 2. 動作確認

ブラウザで以下のURLにアクセスしてください。

**URL:** [http://localhost:3000/sns_2a/](http://localhost:3000/sns_2a/)

## メモ
- サーバーを停止するには、それぞれのターミナルで `Ctrl + C` を押してください。
- バックエンドのログ（APIアクセスやSQLクエリ）は、バックエンド側のターミナルに表示されます。
